# Atlas of Korean Labor — 한국 노동시장 도해

한국고용정보원 KNOW 직업분류의 **537개 직업**과 통계청·고용노동부 공식 통계, 커리어넷 직업백과의 **478개 실측 임금·직무 설명**을 결합해 한국 노동시장 약 **2,720만 개의 일자리**를 한 화면에 펼치는 데이터 탐색 도구입니다.

라이브 데모: <https://jeonsworld.github.io/ai-jobs-ko/>

## 어떻게 쓰는가

- 첫 화면은 **35개 카테고리 트리맵**. 사각형 면적은 취업자 수, 색은 선택한 지표(디지털 AI 노출도 / 평균 연봉 / 5년 전망 / 학력요건).
- 카테고리를 클릭하면 그 안의 **개별 직업이 펼쳐집니다**.
- 우측 사이드바에서 직업명을 검색하거나 카테고리 리스트를 탐색할 수 있습니다.
- 직업 박스 위로 마우스를 올리면 **상세 정보 카드**가 뜹니다 (취업자 / 연봉 / 월임금 환산 / 5년 전망 / 학력 / AI 노출도 / 직무 설명).
- 직업을 클릭하면 커리어넷 직업백과 페이지로 이동합니다.

## 방법론과 한계

- 직업별 취업자수는 KSCO 대분류 공식 통계를 직업명·카테고리 가중치로 분배한 **추정값**이며, 카테고리 합계는 KOSIS 통계와 정합합니다.
- 평균 연봉은 매핑된 298개 직업은 커리어넷 **실측값**, 나머지는 추정값입니다 (툴팁에 [실측]/[추정] 뱃지로 표시).
- AI 노출도는 거친 추정치이며 정밀 예측이 아닙니다. 노출도가 높다고 해서 그 직업이 사라진다는 뜻은 아닙니다.

## 로컬 실행

```bash
# 데이터 빌드
python3 build_data.py

# 정적 파일 서빙
cd site && python3 -m http.server 8765
open http://localhost:8765/index.html
```

## 빌드 파이프라인

```
KEIS KNOW 직업분류 (537개)        ─┐
KOSIS 직업별 취업자 (대분류)        ├─→ build_data.py ─→ site/data.json ─→ site/index.html
고용노동부 임금 통계 (대분류)        │                                       (트리맵 + 사이드 패널)
커리어넷 직업백과 API (552개)        ─┘
```
