# 원티드 × ai-jobs-ko 아이데이션

> ai-jobs-ko가 가진 거시 자산(537개 직업 마스터·AI 노출도 프레임워크·드릴다운 시각화)을 원티드의 미시 자산(공고·이력서·합격·연봉 협상 데이터)과 결합해 만들 수 있는 제품·기능 후보.

## 1. 자산 매핑

| | ai-jobs-ko | 원티드 |
| --- | --- | --- |
| 직업 분류 | KSCO·KNOW 537개 (한국 전체) | 자체 직무 태그 (IT·스타트업 편향) |
| 임금 | KOSIS·고용노동부 대분류 평균 + 커리어넷 478개 실측 | 매칭 풀의 **실제 협상 연봉**·상한·하한 |
| 직업 인구 | 통계청 경활인구 직업별 취업자 (2,720만) | 원티드 풀의 **수요/공급** 실시간 |
| 학력 | 직업명 휴리스틱 추정 | 이력서의 **실제** 학력·전공 분포 |
| 전망 | 5년 휴리스틱 + KEIS 텍스트 | 합격 funnel·재이동 코호트로 **검증 가능한 미래** |
| AI 노출도 | 디지털 업무 비중 0~10점 | — (원티드 + LLM으로 정교화 여지) |
| 시각화 | 트리맵·드릴다운·툴팁 | 매칭 카드·검색 UX |

**핵심 시너지**: 거시(전체 한국 노동시장) ↔ 미시(원티드 풀의 실제 흐름) 비교가 가능. ai-jobs-ko는 *맥락*, 원티드는 *액션*을 제공.

## 2. 아이디어 카드

표기: **데이터** = 어떤 데이터를 쓰는가, **가치** = 사용자/회사가 얻는 것, **난이도** = S/M/L, **임팩트** = ★~★★★.

### A. 구직자 (B2C)

#### A1. AI Career Compass — 노출도 기반 진로 진단
> "당신의 직무는 AI 노출도 8점입니다. 같은 노출도 그룹의 30대 백엔드 개발자 중 23%가 1년 안에 ML 엔지니어로 이동했습니다."

- **데이터**: 이력서 → 직업 매칭(537 분류) + 노출도 점수 + 합격/이직 코호트
- **가치**: 단순 매칭이 아니라 *방향성* 제공. 안전 vs 성장 vs AI 친화 트랙 분기
- **난이도**: M (매칭은 LLM 임베딩, 코호트는 기존 데이터)
- **임팩트**: ★★★ (B2C 차별화·바이럴 가능, 원티드 매거진/콘텐츠 시너지)

#### A2. Career Pathway Atlas — 직업 이동 지도
> 트리맵 위에 사용자의 과거 직무를 점으로 찍고, 같은 출발점 사람들이 어디로 갔는지 화살표로 시각화.

- **데이터**: 이력서의 시계열 직무 변화 + 합격 데이터로 강조
- **가치**: "내 다음 직업"을 데이터 기반으로 제시. 멘토 컨텐츠 대체
- **난이도**: M (Sankey + 트리맵 결합)
- **임팩트**: ★★★ (강력한 PR 임팩트, 원티드만 가능)

#### A3. 협상 레버리지 카드
> 사용자의 학력·경력으로 시장 분포를 보여주고, 비슷한 합격자의 실제 연봉 분포를 노출.

- **데이터**: 합격 시 협상 연봉 + 이력서 매칭
- **가치**: 협상 정보 비대칭 해소 (B2C 핵심 페인포인트)
- **난이도**: S (이미 부분 보유, ai-jobs-ko의 시장 평균과 합치면 풍부)
- **임팩트**: ★★★ (premium 멤버십 가치 명확)

#### A4. Stay or Pivot — 5년 시뮬레이터
> 현재 직업 유지 시 vs 노출도 4 이하 직무로 전환 시 vs 노출도 9 직무로 전환 시 — 5년 후 임금/수요 시뮬레이션.

- **데이터**: ai-jobs-ko 노출도·전망 + 원티드 공고 추세
- **가치**: 직관적 의사결정 도구. 노출 높은 직업도 *변형*되어 살아남는다는 걸 보여주는 교육적 가치
- **난이도**: M
- **임팩트**: ★★ (콘텐츠형, 직접 매출 X)

#### A5. 이력서 → 직업 분류 자동 진단
> 이력서 업로드 시 KSCO/KNOW 537 직업 중 가장 가까운 3~5개 매칭 + AI 노출도 + 보강 스킬 추천.

- **데이터**: 이력서 + 537 직업 마스터 + LLM
- **가치**: 입사지원 추천의 정확도/설명력 향상. 기존 추천 엔진 보완
- **난이도**: S (LLM 임베딩 매칭)
- **임팩트**: ★★ (existing flow 강화)

### B. 기업 (B2B)

#### B1. Talent Atlas Pro — 채용 시장 인텔리전스
> "당신이 채용하는 백엔드 엔지니어는 한국에 X만 명, 원티드 풀에 Y만 명, 평균 협상 연봉 Z만원, 동종업계 대비 +12%."

- **데이터**: 거시 통계 + 원티드 풀 + 경쟁사 공고 가격
- **가치**: HR/Recruiter가 직접 사용. 시장 dashboard
- **난이도**: M (시각화는 ai-jobs-ko 그대로 재활용)
- **임팩트**: ★★★ (B2B 매출 직결, 가격 정당화 도구)

#### B2. Workforce Risk Score — 인력 구성 AI 리스크 진단
> 회사의 인력 50%가 노출 8점 이상이면 "재교육 우선순위 X"로 자동 권고.

- **데이터**: 회사 인력 직무(원티드 채용 기록 + ATS 연동) + ai-jobs-ko 노출도
- **가치**: HR 컨설팅·재교육 사업 연결. ESG·CSR 보고서에도 활용
- **난이도**: L (기업별 맞춤·privacy)
- **임팩트**: ★★★ (대기업 솔루션 매출, 신규 segment)

#### B3. 공고 자동 분류·정합성 검증
> 새 공고 작성 시 537 직업 중 가장 가까운 3개 자동 추천 + 카테고리 트리맵 위에 자사 공고 위치 표시.

- **데이터**: 공고 텍스트 + 537 분류 + LLM
- **가치**: 검색 정확도·매칭 품질 향상. HR이 직무 정의에 어려움 겪을 때 가이드
- **난이도**: S
- **임팩트**: ★★ (matching funnel 개선)

#### B4. 인재 풀 vs 시장 비교
> "당신이 채용하는 직무는 한국에 137만 명, 그중 원티드 풀에 4.8만 명. 풀 점유율 3.5%."

- **데이터**: 거시 + 원티드 미시
- **가치**: 채용 어려움의 *원인 진단* (시장이 작음 vs 원티드에 안 옴 vs 가격 미스매치)
- **난이도**: S
- **임팩트**: ★★

### C. 데이터 프로덕트·브랜드

#### C1. AI Job Atlas (한국판 karpathy/jobs)
> 분기별·연간 갱신되는 권위 있는 한국 노동시장 시각화. 언론·정부·연구기관이 인용하는 단일 소스.

- **데이터**: ai-jobs-ko 그대로 + 원티드 트렌드 오버레이
- **가치**: 브랜드/PR. "원티드 = 한국 노동 데이터의 표준" 포지셔닝
- **난이도**: S (이미 60% 완성)
- **임팩트**: ★★★ (마케팅 가치, B2B 영업 자료)

#### C2. AI 노출도 모델 v2 — 합격 데이터로 검증된 점수
> 현재 휴리스틱 노출도 점수를 *원티드 합격 funnel + 채용 수요 변화*로 재학습. 학계·정책 협력 채널 확보.

- **데이터**: 노출도 라벨 → 직업별 채용 수요 변화 추적 → 라벨 검증/재학습
- **가치**: 학계 논문·OECD 자문 같은 협력 가능. 채용 트렌드 예측 모델로 확장
- **난이도**: L (모델·계약·연구 리소스)
- **임팩트**: ★★★ (장기 자산, 데이터 모트)

#### C3. 분기 리포트: "AI 시대 한국 직업 지도"
> 분기마다 한 페이지 데이터 스토리 발행. 미디어 PR + 채용 영업 leave-behind.

- **데이터**: ai-jobs-ko 갱신 + 원티드 hot/cold 직무 시계열
- **가치**: 콘텐츠 마케팅, 정기적 reach
- **난이도**: S
- **임팩트**: ★★

### D. 내부 ML 운영

#### D1. 매칭 추천에 "직업 거리·노출도" feature 추가
> 537 분류상 거리 + 노출도 차이를 매칭/추천 모델 입력 변수로 추가.

- **데이터**: 추천 모델 학습 데이터 + ai-jobs-ko feature
- **가치**: 매칭 품질 ↑ (가설: 노출도 비슷한 직업끼리 cross-apply 성공률 높음)
- **난이도**: S (feature 추가, A/B 테스트)
- **임팩트**: ★★ (CTR/지원율 KPI 직접 영향)

#### D2. 합격 예측 모델 강화
> 합격 확률에 거시 변수(직업의 AI 노출도, 5년 전망, 시장 인구) 추가.

- **데이터**: 합격 라벨 + 거시 feature
- **가치**: 추천 sort 정확도, 합격축하금 리스크 관리
- **난이도**: S~M
- **임팩트**: ★★

#### D3. 채용 시장 조기 경보
> 노출 8+ 직무 공고 수의 4주 연속 감소 등 거시 패턴 자동 감지 → 프로덕트 팀에 알림.

- **데이터**: 공고 시계열 + 노출도
- **가치**: 트렌드 변화에 빠른 대응. PR 콘텐츠 소재 자동 발굴
- **난이도**: S (Anomaly detection)
- **임팩트**: ★★

## 4. 빠른 실험 (1~2주 단위로 검증 가능)

### 실험 1: 매칭 모델에 노출도 feature 효과 측정
- 가설: 같은 매칭 점수라도 *AI 노출도 거리가 가까운* 추천이 지원율 더 높다
- 방법: D1 feature 추가, A/B 1% 트래픽
- KPI: 지원 클릭률, 합격률
- 기간: 1주 학습 + 2주 노출

### 실험 2: 이력서 진단 카드의 결과 페이지 노출
- 가설: 이력서 업로드 직후 "당신의 직업 분류 + AI 노출도" 카드를 보여주면 페이지 체류·재방문 ↑
- 방법: A5 모듈 만들고 이력서 페이지에 삽입
- KPI: DAU, 재방문률, 매칭 클릭률
- 기간: 2주 빌드 + 2주 측정

### 실험 3: ai-jobs-ko 페이지를 원티드 매거진/블로그 콘텐츠로 publish
- 가설: 데이터 시각화 콘텐츠가 원티드 도메인 SEO·organic traffic 견인
- 방법: jeonsworld.github.io/ai-jobs-ko를 wanted.co.kr/insights/atlas 같은 경로로 마이그레이션 + 분기 리포트(C3) 동봉
- KPI: 페이지 도달, 가입 전환, PR 노출
- 기간: 1~2주
